# Determinants: Meaning Before Formulas

The determinant is often introduced as a mysterious formula involving sums over permutations. We take a different approach: we start with the **properties** a determinant must satisfy, show these properties uniquely determine the function, and then derive consequences. The formulas emerge naturally from the axioms.

## The Determinant: Axiomatic Approach

We seek a function $\det : M_{n \times n}(F) \to F$ that captures whether a matrix is invertible and measures how it scales volume. We define it by its properties.

<Callout type="definition" title="Determinant (Axiomatic Definition)">
  The **determinant** is the unique function $\det : M_{n \times n}(F) \to F$ satisfying:
  1. **Multilinearity in rows:** $\det$ is linear in each row when the other rows are held fixed. If the $i$-th row of $A$ is $\alpha r + r'$ while all other rows are unchanged, then $\det(A) = \alpha \det(A_r) + \det(A_{r'})$, where $A_r$, $A_{r'}$ have row $i$ replaced by $r$, $r'$ respectively.
  2. **Alternating:** If two rows of $A$ are equal, then $\det(A) = 0$.
  3. **Normalization:** $\det(I_n) = 1$.
</Callout>

<Callout type="tip" title="Why These Axioms?">
  These three properties encode geometric intuition about oriented volume:
  - Multilinearity: scaling one edge of a parallelepiped scales volume proportionally, and the volume of a "sum" of parallelepipeds splits linearly.
  - Alternating: a parallelepiped with two identical edges is degenerate (has zero volume).
  - Normalization: the unit cube has volume 1.
</Callout>

From these axioms, we can derive everything.

## Consequences of the Axioms

<Callout type="theorem" title="Row Swap Changes Sign">
  If $B$ is obtained from $A$ by swapping two rows, then $\det(B) = -\det(A)$.
</Callout>

**Proof.** Let $A$ have rows $r_1, \ldots, r_n$. Consider the matrix $C$ obtained by replacing both rows $i$ and $j$ (where $i \neq j$) with $r_i + r_j$. By the alternating property (two equal rows), $\det(C) = 0$.

By multilinearity in row $i$:

$$\det(C) = \det(\ldots, r_i, \ldots, r_i + r_j, \ldots) + \det(\ldots, r_j, \ldots, r_i + r_j, \ldots).$$

Expanding multilinearity in row $j$ for each term:

$$0 = \det(\ldots, r_i, \ldots, r_i, \ldots) + \det(\ldots, r_i, \ldots, r_j, \ldots) + \det(\ldots, r_j, \ldots, r_i, \ldots) + \det(\ldots, r_j, \ldots, r_j, \ldots).$$

The first and last terms are zero (alternating property: two equal rows). So:

$$0 = \det(A) + \det(B)$$

where $B$ is $A$ with rows $i$ and $j$ swapped. Hence $\det(B) = -\det(A)$. $\blacksquare$

<Callout type="theorem" title="Effect of Row Operations on the Determinant">
  Let $A$ be an $n \times n$ matrix.
  1. **Swap two rows:** $\det$ changes sign.
  2. **Multiply a row by $\alpha \neq 0$:** $\det$ is multiplied by $\alpha$.
  3. **Add a multiple of one row to another:** $\det$ is unchanged.
</Callout>

**Proof.** (1) Proved above.

(2) By multilinearity in that row: if $B$ has row $i$ equal to $\alpha r_i$ while $A$ has row $i$ equal to $r_i$, then $\det(B) = \alpha \det(A)$.

(3) Let $B$ be obtained from $A$ by replacing row $i$ with $r_i + \alpha r_j$ ($i \neq j$). By multilinearity in row $i$:

$$\det(B) = \det(A) + \alpha \det(A')$$

where $A'$ is $A$ with row $i$ replaced by $r_j$. But $A'$ has two equal rows ($i$ and $j$ are both $r_j$), so $\det(A') = 0$. Hence $\det(B) = \det(A)$. $\blacksquare$

## Determinant and Invertibility

<Callout type="theorem" title="Determinant and Invertibility">
  An $n \times n$ matrix $A$ is invertible if and only if $\det(A) \neq 0$.
</Callout>

**Proof.** Row-reduce $A$ to its reduced row echelon form $R$. The row operations relate the determinants: if $A \xrightarrow{\text{row ops}} R$, then $\det(A) = c \cdot \det(R)$ for some nonzero scalar $c$ (each swap contributes a factor of $-1$, each row scaling by $\alpha$ contributes $1/\alpha$ to relate back, and row additions don't change the determinant).

If $A$ is invertible, then $R = I_n$, so $\det(R) = 1$, hence $\det(A) = c \cdot 1 \neq 0$.

If $A$ is not invertible, then $R$ has a zero row. By multilinearity, $\det(R) = 0$ (the row of zeros can be written as $0 \cdot r$ for any row $r$, so $\det(R) = 0 \cdot \det(\ldots) = 0$). Hence $\det(A) = c \cdot 0 = 0$. $\blacksquare$

## Multiplicativity

<Callout type="theorem" title="The Determinant Is Multiplicative">
  For any $n \times n$ matrices $A$ and $B$:
  $$\det(AB) = \det(A) \det(B).$$
</Callout>

**Proof.** **Case 1:** $A$ is not invertible. Then $AB$ is not invertible (if $AB$ were invertible, then $A$ would be surjective, contradicting non-invertibility in equal dimensions). So $\det(AB) = 0 = \det(A)\det(B)$ since $\det(A) = 0$.

**Case 2:** $A$ is invertible. Define $d(B) = \det(AB) / \det(A)$. We verify that $d$ satisfies the three determinant axioms with respect to the rows of $B$:

*Multilinearity:* If the $i$-th row of $B$ is $\alpha r + r'$, then the $i$-th row of $AB$ is $\alpha (Ar)^T + (Ar')^T$ (here treating rows of $B$ as determining rows of $AB$ through the multiplication). More precisely, $d$ inherits multilinearity from $\det$ applied to $AB$ because left-multiplication by $A$ is linear.

*Alternating:* If $B$ has two equal rows, then $AB$ has two rows that are equal (since left-multiplication preserves the row dependence relation), so $\det(AB) = 0$, hence $d(B) = 0$.

*Normalization:* $d(I) = \det(A \cdot I) / \det(A) = \det(A) / \det(A) = 1$.

By uniqueness of the determinant, $d(B) = \det(B)$, i.e., $\det(AB) / \det(A) = \det(B)$. $\blacksquare$

<Callout type="tip" title="Corollaries of Multiplicativity">
  Several important facts follow immediately:
  - $\det(A^{-1}) = 1 / \det(A)$ (take $B = A^{-1}$ in $\det(AB) = \det(A)\det(B)$).
  - $\det(A^n) = (\det(A))^n$.
  - If $B = P^{-1}AP$ (similar matrices), then $\det(B) = \det(P^{-1})\det(A)\det(P) = \det(A)$. So the determinant is a similarity invariant.
</Callout>

## The Determinant for Small Matrices

For $1 \times 1$: $\det(a) = a$.

For $2 \times 2$: From the axioms, one can derive

$$\det\begin{pmatrix} a & b \\ c & d \end{pmatrix} = ad - bc.$$

**Derivation.** Expand using multilinearity in row 1: the first row $(a, b) = a(1, 0) + b(0, 1) = ae_1^T + be_2^T$. So:

$$\det\begin{pmatrix} a & b \\ c & d \end{pmatrix} = a\det\begin{pmatrix} 1 & 0 \\ c & d \end{pmatrix} + b\det\begin{pmatrix} 0 & 1 \\ c & d \end{pmatrix}.$$

Now expand each in row 2:

$$= a\left[c\det\begin{pmatrix} 1 & 0 \\ 1 & 0 \end{pmatrix} + d\det\begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}\right] + b\left[c\det\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} + d\det\begin{pmatrix} 0 & 1 \\ 0 & 1 \end{pmatrix}\right].$$

Using the alternating property (equal rows give 0) and normalization ($\det(I) = 1$), and noting $\det\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} = -\det(I) = -1$:

$$= a[0 + d \cdot 1] + b[c \cdot (-1) + 0] = ad - bc.$$

## Geometric Interpretation

In $\mathbb{R}^n$, $|\det(A)|$ equals the $n$-dimensional volume of the parallelepiped spanned by the rows (or columns) of $A$. The sign of $\det(A)$ indicates **orientation**: positive means the rows form a right-handed system (same orientation as the standard basis), negative means left-handed.

For a $2 \times 2$ matrix, $|\det(A)|$ is the area of the parallelogram spanned by its row vectors. For $3 \times 3$, it is the volume of the parallelepiped.

<Problem title="Prove: Determinant of a Triangular Matrix">
  Let $A$ be an upper triangular $n \times n$ matrix (all entries below the diagonal are zero). Prove that $\det(A) = a_{11} a_{22} \cdots a_{nn}$, the product of the diagonal entries.
<Solution>
  **Proof.** We proceed by induction on $n$.
  *Base case ($n = 1$):* $\det(a_{11}) = a_{11}$. $\checkmark$
  *Inductive step:* Assume the result for $(n-1) \times (n-1)$ upper triangular matrices. Let $A$ be $n \times n$ upper triangular.
  **Case 1:** Some diagonal entry $a_{kk} = 0$. Then row $k$ of $A$ is a linear combination of $e_1^T, \ldots, e_{k-1}^T$ (the nonzero entries in row $k$ are in columns $1, \ldots, k-1$, and all entries in column $k$ at or below row $k$ are zero). By row reduction without changing the determinant (adding multiples of rows), we can create two equal rows, giving $\det(A) = 0 = a_{11} \cdots a_{nn}$.
  More directly: if any $a_{kk} = 0$, the matrix is not invertible (the columns are linearly dependent since column $k$ is a combination of earlier columns in an upper triangular matrix with a zero on the diagonal). So $\det(A) = 0$.
  **Case 2:** All diagonal entries are nonzero. Use row operations of type 3 (add a multiple of one row to another, which preserves the determinant) to eliminate all entries above the diagonal. This reduces $A$ to the diagonal matrix $D = \text{diag}(a_{11}, \ldots, a_{nn})$ without changing the determinant.
  For the diagonal matrix: by multilinearity, factor out $a_{ii}$ from each row $i$:
  $$\det(D) = a_{11} a_{22} \cdots a_{nn} \det(I_n) = a_{11} a_{22} \cdots a_{nn}.$$
  Since $\det(A) = \det(D)$, the result follows. $\blacksquare$
</Solution>
</Problem>

<Problem title="Prove det(AB) = det(A)det(B) for 2x2 Matrices Directly">
  Verify the multiplicativity of the determinant for $2 \times 2$ matrices by direct computation. Let $A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$ and $B = \begin{pmatrix} e & f \\ g & h \end{pmatrix}$. Compute $\det(AB)$ and $\det(A)\det(B)$ and show they are equal.
<Solution>
  **Solution.** First, compute $AB$:
  $$AB = \begin{pmatrix} ae + bg & af + bh \\ ce + dg & cf + dh \end{pmatrix}.$$
  Now:
  $$\det(AB) = (ae + bg)(cf + dh) - (af + bh)(ce + dg).$$
  Expanding:
  $$= aecf + aedh + bgcf + bgdh - afce - afdg - bhce - bhdg$$
  $$= aedh + bgcf - afdg - bhce$$
  $$= ad(eh - fg) - bc(eh - fg) = (ad - bc)(eh - fg).$$
  And:
  $$\det(A)\det(B) = (ad - bc)(eh - fg).$$
  So $\det(AB) = \det(A)\det(B)$. $\blacksquare$
</Solution>
</Problem>
