# Joint Distributions and Multivariate Random Variables

So far we have studied random variables one at a time. Real problems involve multiple interacting quantities. Joint distributions encode how random variables behave together, and the key structural question is: when does the joint factor into a product of marginals?

## Joint PMFs and PDFs

<Callout type="definition">
**Definition 10.1 (Joint PMF).** For discrete random variables $X, Y$, the **joint PMF** is

$$p_{X,Y}(x, y) = P(X = x, Y = y).$$

It satisfies $p_{X,Y}(x,y) \geq 0$ and $\sum_x \sum_y p_{X,Y}(x,y) = 1$.
</Callout>

<Callout type="definition">
**Definition 10.2 (Joint PDF).** For continuous random variables $X, Y$, a **joint PDF** $f_{X,Y}$ satisfies $f_{X,Y}(x,y) \geq 0$ and

$$P\bigl((X,Y) \in A\bigr) = \iint_A f_{X,Y}(x,y)\,dx\,dy$$

for all (nice) sets $A \subseteq \mathbb{R}^2$. The normalization condition is $\int_{-\infty}^{\infty}\int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dx\,dy = 1$.
</Callout>

## Marginal Distributions

<Callout type="theorem">
**Theorem 10.3 (Marginals from Joint).** The marginal distributions are obtained by summing (discrete) or integrating (continuous) out the other variable:

$$p_X(x) = \sum_y p_{X,Y}(x,y), \qquad f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dy.$$

*Proof (continuous case).* $P(X \leq a) = P(X \leq a, Y \in \mathbb{R}) = \int_{-\infty}^a \int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dy\,dx$. Differentiating in $a$ gives $f_X(a) = \int_{-\infty}^{\infty} f_{X,Y}(a,y)\,dy$. $\blacksquare$
</Callout>

<Callout type="warning">
**Marginals do not determine the joint.** Many different joint distributions can share the same marginals. The joint contains strictly more information than the pair of marginals.
</Callout>

## Conditional Distributions

<Callout type="definition">
**Definition 10.4 (Conditional Distribution).** For continuous $X, Y$ with $f_Y(y) > 0$:

$$f_{X \mid Y}(x \mid y) = \frac{f_{X,Y}(x, y)}{f_Y(y)}.$$

For discrete $X, Y$ with $P(Y = y) > 0$:

$$P(X = x \mid Y = y) = \frac{p_{X,Y}(x,y)}{p_Y(y)}.$$
</Callout>

This is the natural analog of $P(A \mid B) = P(A \cap B)/P(B)$, extended to random variables.

## Independence and Factorization

<Callout type="theorem">
**Theorem 10.5 (Independence iff Factorization).** Random variables $X$ and $Y$ are independent if and only if the joint distribution factors:

$$f_{X,Y}(x,y) = f_X(x)\,f_Y(y) \qquad \text{for all } x, y$$

(and analogously for PMFs).

*Proof.* $(\Rightarrow)$ Suppose $X \perp Y$. Then for all $a, b$: $P(X \leq a, Y \leq b) = P(X \leq a)\,P(Y \leq b)$, i.e., $F_{X,Y}(a,b) = F_X(a)\,F_Y(b)$. Differentiating with respect to both $a$ and $b$: $f_{X,Y}(a,b) = f_X(a)\,f_Y(b)$.

$(\Leftarrow)$ Suppose $f_{X,Y}(x,y) = f_X(x)f_Y(y)$ for all $x,y$. Then for any $a, b$:

$$P(X \leq a, Y \leq b) = \int_{-\infty}^a\int_{-\infty}^b f_X(x)f_Y(y)\,dy\,dx = \left(\int_{-\infty}^a f_X(x)\,dx\right)\left(\int_{-\infty}^b f_Y(y)\,dy\right)$$

$= P(X \leq a)\,P(Y \leq b)$. Since this holds for all $a, b$, $X \perp Y$. $\blacksquare$
</Callout>

<Callout type="theorem">
**Theorem 10.6.** If $X \perp Y$ and $E[|X|], E[|Y|] < \infty$, then $E[XY] = E[X]\,E[Y]$.

*Proof (continuous case).* By independence, $f_{X,Y}(x,y) = f_X(x)f_Y(y)$. Then

$$E[XY] = \int\!\!\int xy\, f_{X,Y}(x,y)\,dx\,dy = \int\!\!\int xy\, f_X(x)f_Y(y)\,dx\,dy$$

$$= \left(\int x\,f_X(x)\,dx\right)\left(\int y\,f_Y(y)\,dy\right) = E[X]\,E[Y],$$

where the factorization of the double integral uses Fubini's theorem (justified by the absolute integrability assumption). $\blacksquare$
</Callout>

## Convolution Formula

<Callout type="theorem">
**Theorem 10.7 (Convolution).** If $X$ and $Y$ are independent continuous random variables and $Z = X + Y$, then the PDF of $Z$ is

$$f_Z(z) = \int_{-\infty}^{\infty} f_X(x)\,f_Y(z - x)\,dx = (f_X * f_Y)(z).$$

*Proof.* By LOTUS on the joint:

$$F_Z(z) = P(X + Y \leq z) = \int\!\!\int_{x+y \leq z} f_X(x)f_Y(y)\,dx\,dy = \int_{-\infty}^{\infty} f_X(x)\left(\int_{-\infty}^{z-x} f_Y(y)\,dy\right)dx.$$

Differentiating in $z$: $f_Z(z) = \int_{-\infty}^{\infty} f_X(x)\,f_Y(z-x)\,dx$. $\blacksquare$
</Callout>

## E[XY] = E[X]E[Y] Does Not Imply Independence

<Callout type="warning">
**Counterexample 10.8.** Let $X \sim N(0,1)$ and $Y = X^2$. Then $E[XY] = E[X^3] = 0$ (odd moment of a symmetric distribution) and $E[X]E[Y] = 0 \cdot 1 = 0$. So $E[XY] = E[X]E[Y]$.

Yet $X$ and $Y = X^2$ are obviously dependent: $Y$ is a deterministic function of $X$. Even $\text{Cov}(X,Y) = 0$, confirming that uncorrelated random variables need not be independent.
</Callout>

<Callout type="tip">
**When does uncorrelated imply independent?** For jointly Normal random variables $(X, Y)$, uncorrelatedness *does* imply independence. This is a special property of the Normal distribution, not a general fact.
</Callout>

<Problem>
**Exercise 10.1.** Let $(X, Y)$ have joint PDF $f_{X,Y}(x,y) = 6(1-y)$ for $0 \leq x \leq y \leq 1$ and $0$ otherwise. Compute the marginal PDFs $f_X(x)$ and $f_Y(y)$, and determine whether $X$ and $Y$ are independent.
</Problem>

<Solution>
**Marginal of $Y$:** For $0 \leq y \leq 1$:

$$f_Y(y) = \int_0^y 6(1-y)\,dx = 6y(1-y).$$

**Marginal of $X$:** For $0 \leq x \leq 1$:

$$f_X(x) = \int_x^1 6(1-y)\,dy = 6\left[y - \frac{y^2}{2}\right]_x^1 = 6\left(\frac{1}{2} - x + \frac{x^2}{2}\right) = 3(1-x)^2.$$

**Independence check:** $f_X(x) \cdot f_Y(y) = 3(1-x)^2 \cdot 6y(1-y) = 18y(1-x)^2(1-y)$. But $f_{X,Y}(x,y) = 6(1-y)$ for $0 \leq x \leq y \leq 1$. These are not equal (e.g., at $x=0, y=1/2$: product $= 18 \cdot \frac{1}{2} \cdot 1 \cdot \frac{1}{2} = 4.5$ while joint $= 3$). Furthermore, the support $\{0 \leq x \leq y \leq 1\}$ is not a rectangle, which already precludes factorization. So $X$ and $Y$ are **not independent**. $\blacksquare$
</Solution>

<Problem>
**Exercise 10.2.** Let $X \sim \text{Uniform}(-1, 1)$ and $Y = X^2$. Show that $\text{Cov}(X, Y) = 0$ but $X$ and $Y$ are not independent. (This is another demonstration that uncorrelated does not imply independent.)
</Problem>

<Solution>
$E[X] = 0$ by symmetry. $E[XY] = E[X \cdot X^2] = E[X^3] = \int_{-1}^1 \frac{x^3}{2}\,dx = 0$ since $x^3$ is odd and the density is symmetric.

Therefore $\text{Cov}(X, Y) = E[XY] - E[X]E[Y] = 0 - 0 \cdot E[Y] = 0$.

However, $X$ and $Y$ are not independent: $Y = X^2$ is a deterministic function of $X$. Formally, $P(Y \leq 1/4 \mid X = 1/3) = 1$ (since $Y = 1/9 \leq 1/4$), but $P(Y \leq 1/4) = P(X^2 \leq 1/4) = P(-1/2 \leq X \leq 1/2) = 1/2 \neq 1$. $\blacksquare$
</Solution>
