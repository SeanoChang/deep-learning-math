# Conditional Expectation

Having developed conditional probability and Bayes' theorem, we now extend conditioning to expectations. The key conceptual leap: $E[X \mid Y = y]$ is a number for each fixed $y$, but $E[X \mid Y]$ is a **random variable** -- a function of $Y$.

## Conditional Expectation as a Function

<Callout type="definition">
**Definition 11.1 (Conditional expectation given an event).** For a discrete random variable $X$ and an event $B$ with $P(B) > 0$:

$$E[X \mid B] = \sum_x x \, P(X = x \mid B).$$

For a continuous random variable with conditional density $f_{X \mid B}$: $E[X \mid B] = \int_{-\infty}^{\infty} x \, f_{X \mid B}(x) \, dx$.
</Callout>

<Callout type="definition">
**Definition 11.2 (Conditional expectation given a random variable).** Let $X, Y$ be jointly distributed random variables.

- **Discrete case:** $E[X \mid Y = y] = \sum_x x \, P(X = x \mid Y = y)$.
- **Continuous case:** $E[X \mid Y = y] = \int_{-\infty}^{\infty} x \, f_{X \mid Y}(x \mid y) \, dx$.

The **conditional expectation** $E[X \mid Y]$ is the random variable $g(Y)$ where $g(y) = E[X \mid Y = y]$.
</Callout>

**Example (Discrete).** Let $X$ be the sum of two dice rolls $D_1, D_2$. Then $E[X \mid D_1 = 3] = 3 + E[D_2] = 3 + 3.5 = 6.5$. The random variable $E[X \mid D_1] = D_1 + 3.5$ is a function of $D_1$.

**Example (Continuous).** Let $(X, Y)$ have joint density $f(x, y) = 2$ on the triangle $\{0 < x < y < 1\}$. Then $f_Y(y) = 2y$ for $y \in (0,1)$, so $f_{X \mid Y}(x \mid y) = 2 / (2y) = 1/y$ for $0 < x < y$. Thus:

$$E[X \mid Y = y] = \int_0^y x \cdot \frac{1}{y} \, dx = \frac{1}{y} \cdot \frac{y^2}{2} = \frac{y}{2}.$$

So $E[X \mid Y] = Y/2$.

## The Law of Total Expectation

<Callout type="theorem">
**Theorem 11.3 (Law of Total Expectation / Tower Property).** For any random variables $X, Y$ with $E[|X|] < \infty$:

$$E[X] = E\!\big[E[X \mid Y]\big].$$
</Callout>

*Proof (discrete case).* Let $g(y) = E[X \mid Y = y]$. Then:

$$E\!\big[E[X \mid Y]\big] = \sum_y g(y) \, P(Y = y) = \sum_y \left(\sum_x x \, P(X = x \mid Y = y)\right) P(Y = y).$$

Substituting $P(X = x \mid Y = y) \cdot P(Y = y) = P(X = x, Y = y)$:

$$= \sum_y \sum_x x \, P(X = x, Y = y) = \sum_x x \sum_y P(X = x, Y = y) = \sum_x x \, P(X = x) = E[X]. \quad \square$$

<Callout type="tip">
**Intuition.** To compute $E[X]$, first compute $E[X \mid Y = y]$ for each $y$ (the "inner" expectation), then average over $Y$ (the "outer" expectation). This is the probabilistic analog of iterated integration.
</Callout>

**Application.** Let $N \sim \text{Poisson}(\lambda)$ and given $N = n$, let $X = \sum_{i=1}^n Z_i$ where $Z_i \sim \text{Bernoulli}(p)$ are i.i.d. Then $E[X \mid N = n] = np$, so $E[X \mid N] = Np$, and by the tower property:

$$E[X] = E[Np] = p \, E[N] = p\lambda.$$

## The Law of Total Variance

<Callout type="theorem">
**Theorem 11.4 (Law of Total Variance / Eve's Law).** For random variables $X, Y$ with $\text{Var}(X) < \infty$:

$$\text{Var}(X) = E\!\big[\text{Var}(X \mid Y)\big] + \text{Var}\!\big(E[X \mid Y]\big).$$
</Callout>

*Proof.* Write $\mu(Y) = E[X \mid Y]$ and $v(Y) = \text{Var}(X \mid Y) = E[X^2 \mid Y] - (E[X \mid Y])^2$. By the tower property applied to $X^2$:

$$E[X^2] = E\!\big[E[X^2 \mid Y]\big] = E\!\big[v(Y) + \mu(Y)^2\big] = E[v(Y)] + E[\mu(Y)^2].$$

Also, $(E[X])^2 = (E[\mu(Y)])^2$ by the tower property. Therefore:

$$\text{Var}(X) = E[X^2] - (E[X])^2 = E[v(Y)] + E[\mu(Y)^2] - (E[\mu(Y)])^2 = E[v(Y)] + \text{Var}(\mu(Y)). \quad \square$$

<Callout type="info">
**Interpretation.** The total variance of $X$ decomposes into:

- $E[\text{Var}(X \mid Y)]$: the **average within-group variance** (unexplained variance),
- $\text{Var}(E[X \mid Y])$: the **between-group variance** (explained variance).

This is the foundation of ANOVA (analysis of variance) in statistics.
</Callout>

## The Best Predictor Theorem

<Callout type="theorem">
**Theorem 11.5 (Best Predictor / Minimum Mean Squared Error).** Among all functions $g(Y)$, the conditional expectation $E[X \mid Y]$ minimizes the mean squared prediction error:

$$E[X \mid Y] = \arg\min_{g} \, E\!\big[(X - g(Y))^2\big].$$
</Callout>

*Proof.* Let $\mu(Y) = E[X \mid Y]$ and let $g(Y)$ be any function of $Y$. Write $X - g(Y) = (X - \mu(Y)) + (\mu(Y) - g(Y))$. Expanding the square:

$$E[(X - g(Y))^2] = E[(X - \mu(Y))^2] + 2E[(X - \mu(Y))(\mu(Y) - g(Y))] + E[(\mu(Y) - g(Y))^2].$$

We claim the cross term vanishes. By the tower property, conditioning on $Y$:

$$E[(X - \mu(Y))(\mu(Y) - g(Y))] = E\!\big[E[(X - \mu(Y))(\mu(Y) - g(Y)) \mid Y]\big].$$

Since $\mu(Y) - g(Y)$ is a function of $Y$, it factors out of the inner expectation:

$$= E\!\big[(\mu(Y) - g(Y)) \cdot E[X - \mu(Y) \mid Y]\big] = E\!\big[(\mu(Y) - g(Y)) \cdot 0\big] = 0$$

because $E[X - \mu(Y) \mid Y] = E[X \mid Y] - \mu(Y) = 0$. Therefore:

$$E[(X - g(Y))^2] = E[(X - \mu(Y))^2] + E[(\mu(Y) - g(Y))^2] \geq E[(X - \mu(Y))^2]$$

with equality iff $g(Y) = \mu(Y) = E[X \mid Y]$ a.s. $\square$

<Callout type="tip">
**Connection to machine learning.** This theorem is why the regression function $E[Y \mid X = x]$ is the optimal predictor under squared loss. Neural networks trained with MSE loss are approximating $E[Y \mid X]$.
</Callout>

## Exercises

<Problem title="Exercise 11.1">
Let $X \sim \text{Uniform}(0, 1)$ and given $X = x$, let $Y \sim \text{Uniform}(0, x)$. Compute $E[Y]$ and $\text{Var}(Y)$ using the laws of total expectation and total variance.
</Problem>

<Solution>
**Computing $E[Y]$:** We have $E[Y \mid X = x] = x/2$, so $E[Y \mid X] = X/2$. By the tower property:

$$E[Y] = E[X/2] = \frac{1}{2} E[X] = \frac{1}{2} \cdot \frac{1}{2} = \frac{1}{4}.$$

**Computing $\text{Var}(Y)$:** Since $Y \mid X = x \sim \text{Uniform}(0, x)$, we have $\text{Var}(Y \mid X = x) = x^2/12$. So $\text{Var}(Y \mid X) = X^2/12$ and $E[Y \mid X] = X/2$.

By the law of total variance:

$$\text{Var}(Y) = E[\text{Var}(Y \mid X)] + \text{Var}(E[Y \mid X]) = E\!\left[\frac{X^2}{12}\right] + \text{Var}\!\left(\frac{X}{2}\right).$$

Now $E[X^2] = \int_0^1 x^2 \, dx = 1/3$ and $\text{Var}(X/2) = \text{Var}(X)/4 = (1/12)/4 = 1/48$. Thus:

$$\text{Var}(Y) = \frac{1/3}{12} + \frac{1}{48} = \frac{1}{36} + \frac{1}{48} = \frac{4 + 3}{144} = \frac{7}{144}. \quad \square$$
</Solution>

<Problem title="Exercise 11.2">
Prove that if $X$ and $Y$ are independent, then $E[X \mid Y] = E[X]$ (a constant random variable). Deduce that the law of total variance reduces to $\text{Var}(X) = E[\text{Var}(X \mid Y)]$ in this case.
</Problem>

<Solution>
**Part 1.** If $X \perp Y$, then $f_{X \mid Y}(x \mid y) = f_X(x)$ (or $P(X = x \mid Y = y) = P(X = x)$ in the discrete case). Therefore:

$$E[X \mid Y = y] = \int x \, f_{X \mid Y}(x \mid y) \, dx = \int x \, f_X(x) \, dx = E[X]$$

for every $y$. Since $g(y) = E[X]$ is constant, $E[X \mid Y] = E[X]$ a.s.

**Part 2.** When $E[X \mid Y] = E[X]$ is constant, $\text{Var}(E[X \mid Y]) = \text{Var}(\text{constant}) = 0$. The law of total variance gives:

$$\text{Var}(X) = E[\text{Var}(X \mid Y)] + 0 = E[\text{Var}(X \mid Y)].$$

Moreover, since $\text{Var}(X \mid Y = y) = E[X^2 \mid Y = y] - (E[X])^2$ and independence gives $E[X^2 \mid Y = y] = E[X^2]$, we get $\text{Var}(X \mid Y) = \text{Var}(X)$ a.s., so $E[\text{Var}(X \mid Y)] = \text{Var}(X)$, which is consistent. $\square$
</Solution>
