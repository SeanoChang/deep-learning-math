# Numerical Integration and Error Analysis

While the Fundamental Theorem of Calculus gives us a powerful method for evaluating integrals, it requires finding an explicit antiderivative -- which is often impossible. In practice, we approximate definite integrals numerically. The key question then becomes: **how accurate is the approximation?** In this lesson we derive the major numerical integration rules and establish rigorous error bounds.

## Riemann Sums Revisited

Recall that for a Riemann-integrable function $f$ on $[a,b]$, the integral is the limit of Riemann sums:

$$\int_a^b f(x)\,dx = \lim_{\|P\| \to 0} \sum_{k=1}^n f(c_k) \Delta x_k.$$

For a uniform partition with $n$ subintervals of width $h = (b-a)/n$, the three standard choices of sample point give:

- **Left-endpoint rule:** $L_n = h \sum_{k=0}^{n-1} f(a + kh)$
- **Right-endpoint rule:** $R_n = h \sum_{k=1}^{n} f(a + kh)$
- **Midpoint rule:** $M_n = h \sum_{k=1}^{n} f\!\left(a + \left(k - \tfrac{1}{2}\right)h\right)$

All three converge to the integral as $n \to \infty$, but they converge at different rates.

## The Trapezoidal Rule

<Callout type="definition">
**Definition 8.1 (Trapezoidal Rule).** For a uniform partition of $[a,b]$ into $n$ subintervals of width $h = (b-a)/n$, the **trapezoidal approximation** is

$$T_n = \frac{h}{2}\bigl[f(x_0) + 2f(x_1) + 2f(x_2) + \cdots + 2f(x_{n-1}) + f(x_n)\bigr]$$

where $x_k = a + kh$.
</Callout>

Observe that $T_n = \frac{L_n + R_n}{2}$: the trapezoidal rule is the average of the left and right endpoint rules. Geometrically, it replaces the graph of $f$ on each subinterval by a straight line segment, forming trapezoids.

<Callout type="theorem">
**Theorem 8.2 (Trapezoidal Error Bound).** Let $f \in C^2[a,b]$ (i.e., $f''$ exists and is continuous on $[a,b]$). Then the error in the trapezoidal rule satisfies

$$\left|\int_a^b f(x)\,dx - T_n\right| \leq \frac{(b-a)^3}{12n^2} \cdot \max_{x \in [a,b]} |f''(x)|.$$

In particular, the trapezoidal rule has **order 2** convergence: the error is $O(h^2)$ as $n \to \infty$.
</Callout>

*Proof.* Let

$$M_2=\max_{x\in[a,b]}|f''(x)|,$$

which exists because $f''$ is continuous on $[a,b]$.

Fix one subinterval $[x_{k-1},x_k]$ of width $h$, and let $p_k$ be the linear interpolant of $f$ at the endpoints.
Then the trapezoidal contribution on this subinterval is exactly

$$T_k=\int_{x_{k-1}}^{x_k} p_k(x)\,dx,$$

so the local error is

$$E_k=\int_{x_{k-1}}^{x_k}\bigl(f(x)-p_k(x)\bigr)\,dx.$$

For linear interpolation, the remainder formula gives: for each $x\in[x_{k-1},x_k]$,

$$f(x)-p_k(x)=\frac{f''(\xi_x)}{2}(x-x_{k-1})(x-x_k)$$

for some $\xi_x\in(x_{k-1},x_k)$.

Because $f''$ is continuous, the standard trapezoidal-error form implies there exists
$\zeta_k\in(x_{k-1},x_k)$ such that

$$E_k=\frac{f''(\zeta_k)}{2}\int_{x_{k-1}}^{x_k}(x-x_{k-1})(x-x_k)\,dx.$$

Now set $t=x-x_{k-1}$, so $t\in[0,h]$ and

$$\int_{x_{k-1}}^{x_k}(x-x_{k-1})(x-x_k)\,dx
=\int_0^h t(t-h)\,dt
=-\frac{h^3}{6}.$$

Hence

$$E_k=-\frac{h^3}{12}f''(\zeta_k), \qquad |E_k|\le \frac{h^3}{12}M_2.$$

Summing over all $n$ subintervals:

$$|E_T|=\left|\sum_{k=1}^n E_k\right|
\le \sum_{k=1}^n |E_k|
\le n\cdot \frac{h^3}{12}M_2
=\frac{(b-a)h^2}{12}M_2
=\frac{(b-a)^3}{12n^2}M_2.$$

So

$$\left|\int_a^b f(x)\,dx-T_n\right|
\le \frac{(b-a)^3}{12n^2}\max_{x\in[a,b]}|f''(x)|.$$

Therefore the trapezoidal rule is $O(h^2)$. $\square$

## The Midpoint Rule

<Callout type="theorem">
**Theorem 8.3 (Midpoint Error Bound).** Under the same hypotheses as Theorem 8.2,

$$\left|\int_a^b f(x)\,dx - M_n\right| \leq \frac{(b-a)^3}{24n^2} \cdot \max_{x \in [a,b]} |f''(x)|.$$
</Callout>

<Callout type="info">
**Remark.** The midpoint rule has the *same order* of convergence as the trapezoidal rule ($O(h^2)$), but its error constant is half as large. This is because the midpoint rule benefits from symmetry: odd-order error terms cancel.
</Callout>

## Simpson's Rule

<Callout type="definition">
**Definition 8.4 (Simpson's Rule).** For an **even** number $n$ of subintervals with $h = (b-a)/n$, **Simpson's rule** is

$$S_n = \frac{h}{3}\bigl[f(x_0) + 4f(x_1) + 2f(x_2) + 4f(x_3) + 2f(x_4) + \cdots + 4f(x_{n-1}) + f(x_n)\bigr].$$

Equivalently, $S_n = \frac{2M_{n/2} + T_n}{3}$ (a weighted average of the midpoint and trapezoidal rules, with weight $2/3$ on the midpoint).
</Callout>

On each pair of subintervals $[x_{2k}, x_{2k+2}]$, Simpson's rule fits a quadratic through the three points $(x_{2k}, f(x_{2k}))$, $(x_{2k+1}, f(x_{2k+1}))$, $(x_{2k+2}, f(x_{2k+2}))$ and integrates the quadratic exactly.

<Callout type="theorem">
**Theorem 8.5 (Simpson's Error Bound).** Let $f \in C^4[a,b]$. Then

$$\left|\int_a^b f(x)\,dx - S_n\right| \leq \frac{(b-a)^5}{180n^4} \cdot \max_{x \in [a,b]} |f^{(4)}(x)|.$$

Simpson's rule has **order 4** convergence: the error is $O(h^4)$.
</Callout>

<Callout type="info">
**Why order 4?** Simpson's rule is exact for polynomials of degree $\leq 3$, even though it is constructed from quadratic interpolation. This is because the cubic error term vanishes by symmetry on each pair of subintervals. The first nonzero error term involves $f^{(4)}$.
</Callout>

## Summary of Error Orders

| Method | Error Bound | Order |
|---|---|---|
| Left/Right endpoint | $\frac{(b-a)^2}{2n} \max \lvert f' \rvert$ | $O(h)$ |
| Trapezoidal | $\frac{(b-a)^3}{12n^2} \max \lvert f'' \rvert$ | $O(h^2)$ |
| Midpoint | $\frac{(b-a)^3}{24n^2} \max \lvert f'' \rvert$ | $O(h^2)$ |
| Simpson's | $\frac{(b-a)^5}{180n^4} \max \lvert f^{(4)} \rvert$ | $O(h^4)$ |

The key insight: **smoother functions admit faster approximation**. Each additional degree of smoothness we exploit yields a higher-order method.

## Exercises

<Problem>
**Exercise 8.1.** Prove that the trapezoidal rule is exact for linear functions, i.e., if $f(x) = \alpha x + \beta$, then $T_n = \int_a^b f(x)\,dx$ for any $n$.
</Problem>

<Solution>
*Proof.* Let $f(x) = \alpha x + \beta$ and $h = (b-a)/n$. The trapezoidal rule gives:

$$T_n = \frac{h}{2}\sum_{k=1}^n [f(x_{k-1}) + f(x_k)].$$

Since $f$ is linear, $\frac{f(x_{k-1}) + f(x_k)}{2} = f\!\left(\frac{x_{k-1}+x_k}{2}\right)$, and a linear function on $[x_{k-1}, x_k]$ satisfies

$$\int_{x_{k-1}}^{x_k} f(x)\,dx = f\!\left(\frac{x_{k-1}+x_k}{2}\right) \cdot h = \frac{h}{2}[f(x_{k-1}) + f(x_k)].$$

This is exactly the trapezoidal approximation on each subinterval. Summing over all subintervals:

$$T_n = \sum_{k=1}^n \int_{x_{k-1}}^{x_k} f(x)\,dx = \int_a^b f(x)\,dx.$$

Alternatively, note that $f'' = 0$, so the error bound $\frac{(b-a)^3}{12n^2}\max|f''| = 0$. $\blacksquare$
</Solution>

<Problem>
**Exercise 8.2.** Simpson's rule is exact for cubics. Verify this by showing that $S_2$ gives the exact value of $\int_{-1}^1 x^3\,dx$ (which equals $0$).
</Problem>

<Solution>
With $n = 2$, $a = -1$, $b = 1$, $h = 1$, nodes are $x_0 = -1$, $x_1 = 0$, $x_2 = 1$.

$$S_2 = \frac{h}{3}[f(x_0) + 4f(x_1) + f(x_2)] = \frac{1}{3}[(-1)^3 + 4(0)^3 + (1)^3] = \frac{1}{3}[-1 + 0 + 1] = 0.$$

The exact value is $\int_{-1}^1 x^3\,dx = \left[\frac{x^4}{4}\right]_{-1}^1 = \frac{1}{4} - \frac{1}{4} = 0$.

They agree. This is not a coincidence: Simpson's rule integrates all polynomials of degree $\leq 3$ exactly because the $O(h^4)$ error term involves $f^{(4)}$, which vanishes for cubics. $\blacksquare$
</Solution>

<Problem>
**Exercise 8.3 (Counterexample).** The trapezoidal rule gives the **exact** answer for all linear functions $f(x) = \alpha x + \beta$, regardless of $n$. Verify this for $f(x) = 3x + 2$ on $[0, 4]$ with $n = 1$ (a single trapezoid), and explain why this follows directly from the error bound $\frac{(b-a)^3}{12n^2}\max|f''|$.

Then construct a function for which the trapezoidal rule with $n = 1$ gives a poor approximation, illustrating that the error bound is sharp. Use $f(x) = x^2$ on $[0, 1]$ and compute the exact error.

*Hint:* For a linear function, $f'' = 0$, so the error bound is zero.
</Problem>

<Solution>
**Part 1: Trapezoidal rule is exact for linear functions.**

Let $f(x) = 3x + 2$ on $[0, 4]$ with $n = 1$, $h = 4$. The trapezoidal approximation is:

$$T_1 = \frac{h}{2}[f(0) + f(4)] = \frac{4}{2}[2 + 14] = 2 \cdot 16 = 32.$$

The exact integral: $\int_0^4 (3x + 2)\,dx = \left[\frac{3x^2}{2} + 2x\right]_0^4 = 24 + 8 = 32$.

They agree: $T_1 = 32 = \int_0^4 f$. This is guaranteed by the error bound: $f''(x) = 0$, so $\frac{(b-a)^3}{12n^2}\max|f''| = 0$.

**Part 2: A function where $n = 1$ is poor.**

Let $f(x) = x^2$ on $[0, 1]$ with $n = 1$, $h = 1$:

$$T_1 = \frac{1}{2}[f(0) + f(1)] = \frac{1}{2}[0 + 1] = \frac{1}{2}.$$

The exact integral: $\int_0^1 x^2\,dx = 1/3$.

The error: $|T_1 - 1/3| = |1/2 - 1/3| = 1/6 \approx 0.1667$.

The error bound predicts: $\frac{(1-0)^3}{12 \cdot 1^2} \cdot \max_{[0,1]} |2| = \frac{1}{12} \cdot 2 = \frac{1}{6}$.

The actual error exactly equals the error bound -- the bound is sharp for this function. The trapezoidal rule approximates the parabola $x^2$ by a straight line from $(0,0)$ to $(1,1)$, overestimating the area by exactly $1/6$. $\square$
</Solution>

## Implementing and Comparing Numerical Methods

<PythonPlayground
  title={"Comparing Numerical Integration Methods"}
  defaultCode={`import math

def left_rule(f, a, b, n):
    h = (b - a) / n
    return h * sum(f(a + k*h) for k in range(n))

def trapezoidal(f, a, b, n):
    h = (b - a) / n
    return h * (0.5*f(a) + sum(f(a + k*h) for k in range(1, n)) + 0.5*f(b))

def midpoint(f, a, b, n):
    h = (b - a) / n
    return h * sum(f(a + (k + 0.5)*h) for k in range(n))

def simpsons(f, a, b, n):
    """n must be even."""
    h = (b - a) / n
    s = f(a) + f(b)
    for k in range(1, n):
        s += (4 if k % 2 == 1 else 2) * f(a + k*h)
    return s * h / 3

# Test function: integral of sin(x) from 0 to pi = 2
f = math.sin
a, b = 0, math.pi
exact = 2.0

print("Integral of sin(x) from 0 to pi (exact = 2)")
print(f"{'n':>6} {'Left':>14} {'Trapez':>14} {'Midpoint':>14} {'Simpson':>14}")
print("-" * 66)

for n in [4, 8, 16, 32, 64, 128, 256]:
    L = left_rule(f, a, b, n)
    T = trapezoidal(f, a, b, n)
    M = midpoint(f, a, b, n)
    S = simpsons(f, a, b, n)
    print(f"{n:>6} {abs(L-exact):>14.2e} {abs(T-exact):>14.2e} "
          f"{abs(M-exact):>14.2e} {abs(S-exact):>14.2e}")

print()
print("Observe the convergence rates:")
print("  Left:     error ~ C/n     (halving when n doubles)")
print("  Trapez:   error ~ C/n^2   (quartering when n doubles)")
print("  Midpoint: error ~ C/n^2   (quartering, smaller constant)")
print("  Simpson:  error ~ C/n^4   (1/16 when n doubles)")
`}
