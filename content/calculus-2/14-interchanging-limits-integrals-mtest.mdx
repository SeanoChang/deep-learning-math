# Interchanging Limits and Integrals; The Weierstrass M-test

The central question of this lecture: when can we exchange the order of a limit and an integral? That is, when does

$$\lim_{n \to \infty} \int_a^b f_n(x)\, dx = \int_a^b \lim_{n \to \infty} f_n(x)\, dx ?$$

The answer, as we will see, is that uniform convergence is the key hypothesis that makes this interchange valid.

## The Weierstrass M-test

The $M$-test is the workhorse for establishing uniform convergence of series of functions.

<Callout type="theorem">
**Theorem 14.1 (Weierstrass M-test).** Let $g_n : A \to \mathbb{R}$ be a sequence of functions. Suppose there exist constants $M_n \geq 0$ such that:

1. $|g_n(x)| \leq M_n$ for all $x \in A$ and all $n$,
2. $\sum_{n=1}^{\infty} M_n < \infty$.

Then $\sum_{n=1}^{\infty} g_n$ converges uniformly (and absolutely) on $A$.
</Callout>

*Proof.* We use the Cauchy criterion for uniform convergence. Let $\varepsilon > 0$. Since $\sum M_n$ converges, there exists $N$ such that $\sum_{n=m+1}^{k} M_n < \varepsilon$ for all $k > m \geq N$.

For any $x \in A$ and $k > m \geq N$:

$$\left|\sum_{n=m+1}^{k} g_n(x)\right| \leq \sum_{n=m+1}^{k} |g_n(x)| \leq \sum_{n=m+1}^{k} M_n < \varepsilon.$$

This bound is independent of $x$, so

$$\sup_{x \in A} \left|\sum_{n=m+1}^{k} g_n(x)\right| \leq \sum_{n=m+1}^{k} M_n < \varepsilon.$$

By the Cauchy criterion, the partial sums $S_N(x) = \sum_{n=1}^{N} g_n(x)$ converge uniformly on $A$.

For absolute convergence: $\sum |g_n(x)| \leq \sum M_n < \infty$ for each $x$, so $\sum g_n(x)$ converges absolutely at each point. $\square$

<Callout type="tip">
**Strategy for using the M-test.** Given a series $\sum g_n(x)$:
1. Find a bound $|g_n(x)| \leq M_n$ that is independent of $x$ (take the supremum over $x \in A$).
2. Check whether $\sum M_n$ converges (using any test: geometric, $p$-test, etc.).
3. If yes, conclude uniform and absolute convergence on $A$.
</Callout>

## The Limit-Integral Interchange

<Callout type="theorem">
**Theorem 14.2 (Limit-Integral Interchange).** Let $f_n : [a, b] \to \mathbb{R}$ be Riemann integrable for each $n$, and suppose $f_n \to f$ uniformly on $[a, b]$. Then $f$ is Riemann integrable on $[a, b]$ and

$$\lim_{n \to \infty} \int_a^b f_n(x)\, dx = \int_a^b f(x)\, dx.$$

Equivalently, $\int_a^b \lim_{n \to \infty} f_n(x)\, dx = \lim_{n \to \infty} \int_a^b f_n(x)\, dx$.
</Callout>

*Proof.* We first prove integrability of $f$.

Let $\varepsilon > 0$. By uniform convergence, choose $n_0$ such that

$$\|f-f_{n_0}\|_\infty := \sup_{x\in[a,b]} |f(x)-f_{n_0}(x)| < \frac{\varepsilon}{4(b-a)}.$$

Since $f_{n_0}$ is Riemann integrable, there exists a partition $P$ of $[a,b]$ with

$$U(f_{n_0},P)-L(f_{n_0},P) < \frac{\varepsilon}{2}.$$

On each subinterval $I$ of $P$, write $\eta=\|f-f_{n_0}\|_\infty$. Then

$$\sup_I f \le \sup_I f_{n_0} + \eta, \qquad \inf_I f \ge \inf_I f_{n_0} - \eta,$$

so the oscillation satisfies

$$\omega_I(f) \le \omega_I(f_{n_0}) + 2\eta.$$

Summing over $P$:

$$U(f,P)-L(f,P) \le U(f_{n_0},P)-L(f_{n_0},P) + 2\eta(b-a) < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon.$$

By the $\varepsilon$-partition criterion, $f$ is Riemann integrable.

Now prove interchange. Given $\varepsilon > 0$, choose $N$ such that for $n\ge N$,

$$\|f_n-f\|_\infty < \frac{\varepsilon}{b-a}.$$

Then for $n \geq N$:

$$\left|\int_a^b f_n(x)\, dx - \int_a^b f(x)\, dx\right| = \left|\int_a^b [f_n(x) - f(x)]\, dx\right| \leq \int_a^b |f_n(x) - f(x)|\, dx < \frac{\varepsilon}{b - a} \cdot (b - a) = \varepsilon.$$

Therefore $\int_a^b f_n \to \int_a^b f$. $\square$

<Callout type="warning">
**Pointwise convergence is NOT enough!** Consider $f_n : [0, 1] \to \mathbb{R}$ defined by

$$f_n(x) = \begin{cases} n^2 x & \text{if } 0 \leq x \leq 1/n, \\ n^2(2/n - x) & \text{if } 1/n < x \leq 2/n, \\ 0 & \text{if } 2/n < x \leq 1. \end{cases}$$

This is a "spike" of height $n$ and width $2/n$. Then $f_n(x) \to 0$ pointwise for each $x$, but $\int_0^1 f_n = 1$ for every $n$. So the limit of the integrals is $1$, while the integral of the limit is $0$.
</Callout>

## Term-by-Term Integration of Series

<Callout type="theorem">
**Theorem 14.3 (Term-by-Term Integration).** If $\sum g_n$ converges uniformly to $S$ on $[a, b]$ and each $g_n$ is Riemann integrable, then

$$\int_a^b S(x)\, dx = \int_a^b \sum_{n=1}^{\infty} g_n(x)\, dx = \sum_{n=1}^{\infty} \int_a^b g_n(x)\, dx.$$
</Callout>

*Proof.* The partial sums $S_N(x) = \sum_{n=1}^{N} g_n(x)$ converge uniformly to $S(x)$. By Theorem 14.2:

$$\int_a^b S(x)\, dx = \lim_{N \to \infty} \int_a^b S_N(x)\, dx = \lim_{N \to \infty} \sum_{n=1}^{N} \int_a^b g_n(x)\, dx = \sum_{n=1}^{\infty} \int_a^b g_n(x)\, dx. \quad \square$$

## Term-by-Term Differentiation of Power Series

<Callout type="theorem">
**Theorem 14.4 (Term-by-Term Differentiation of Power Series).** Let $f(x) = \sum_{n=0}^{\infty} c_n (x - a)^n$ have radius of convergence $R > 0$. Then $f$ is differentiable on $(a - R, a + R)$ and

$$f'(x) = \sum_{n=1}^{\infty} n c_n (x - a)^{n-1}$$

for all $x \in (a - R, a + R)$. Moreover, the differentiated series has the same radius of convergence $R$.
</Callout>

*Proof.* First compute the radius of convergence of the derived series:

$$R' = \frac{1}{\limsup |nc_n|^{1/n}} = \frac{1}{\limsup |c_n|^{1/n}} = R$$

since $n^{1/n} \to 1$.

Fix $r$ with $0<r<R$, and choose $\rho$ with $r<\rho<R$. Because $\rho<R$, the scalar series
$\sum |c_n|\rho^n$ converges, so the sequence $A_n:=|c_n|\rho^n$ is bounded: $A_n\le C$.

For $|x-a|\le r$:

$$|n c_n (x-a)^{n-1}| \le n|c_n|r^{n-1}
= \frac{n}{r}\left(\frac{r}{\rho}\right)^n A_n
\le \frac{C}{r}\, n q^n,$$

where $q=r/\rho\in(0,1)$. Since $\sum n q^n$ converges, the $M$-test implies
$\sum_{n=1}^\infty n c_n (x-a)^{n-1}$ converges uniformly on $|x-a|\le r$.

Also $\sum c_n(x-a)^n$ converges uniformly on $|x-a|\le r$ (again by $M$-test).
Define partial sums

$$S_N(x)=\sum_{n=0}^N c_n(x-a)^n,\qquad G_N(x)=S_N'(x)=\sum_{n=1}^N n c_n(x-a)^{n-1}.$$

Let $G(x)=\sum_{n=1}^\infty n c_n(x-a)^{n-1}$; by uniform convergence, $G_N\to G$ uniformly on $|x-a|\le r$.
For $x$ in this interval:

$$S_N(x)-S_N(a)=\int_a^x G_N(t)\,dt.$$

Take $N\to\infty$. Uniform convergence of $S_N$ and Theorem 14.2 on the right give

$$f(x)-f(a)=\int_a^x G(t)\,dt.$$

Hence $f$ is differentiable and $f'(x)=G(x)=\sum_{n=1}^\infty n c_n(x-a)^{n-1}$ on $|x-a|<r$.
Since $r<R$ was arbitrary, this holds for all $x\in(a-R,a+R)$. $\square$

<Callout type="info">
**Consequence.** Power series are infinitely differentiable within their radius of convergence, and we can differentiate term-by-term as many times as we like. This is a remarkable property not shared by general uniformly convergent series.
</Callout>

## Application: $e^x$ from its Power Series

<Callout type="definition">
**Example 14.5.** The exponential function $e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}$ has $R = \infty$. By term-by-term differentiation:

$$\frac{d}{dx} e^x = \sum_{n=1}^{\infty} \frac{n x^{n-1}}{n!} = \sum_{n=1}^{\infty} \frac{x^{n-1}}{(n-1)!} = \sum_{m=0}^{\infty} \frac{x^m}{m!} = e^x.$$

This proves the famous identity $(e^x)' = e^x$ directly from the power series.
</Callout>

## Exercises

<Problem>
**Exercise 14.1.** Prove the Weierstrass $M$-test: if $|g_n(x)| \leq M_n$ for all $x \in A$ and $\sum M_n < \infty$, then $\sum g_n$ converges uniformly on $A$.
</Problem>

<Solution>
*Proof.* We show the partial sums $S_N(x) = \sum_{n=1}^{N} g_n(x)$ are uniformly Cauchy, which implies uniform convergence (by the Cauchy criterion for uniform convergence).

Let $\varepsilon > 0$. Since $\sum_{n=1}^{\infty} M_n$ converges, the partial sums $\sum_{n=1}^{N} M_n$ form a Cauchy sequence in $\mathbb{R}$. Hence there exists $N_0$ such that for all $k > m \geq N_0$:

$$\sum_{n=m+1}^{k} M_n < \varepsilon.$$

Now for any $x \in A$ and $k > m \geq N_0$:

$$|S_k(x) - S_m(x)| = \left|\sum_{n=m+1}^{k} g_n(x)\right| \leq \sum_{n=m+1}^{k} |g_n(x)| \leq \sum_{n=m+1}^{k} M_n < \varepsilon.$$

The bound $\varepsilon$ is independent of $x$, so $\sup_{x \in A} |S_k(x) - S_m(x)| < \varepsilon$ for $k > m \geq N_0$.

By the Cauchy criterion (Theorem 13.6), the series $\sum g_n$ converges uniformly on $A$.

For absolute convergence at each point: $\sum |g_n(x)| \leq \sum M_n < \infty$, so the series converges absolutely. $\square$
</Solution>

<Problem>
**Exercise 14.2.** Show that $\sum_{n=1}^{\infty} \frac{\cos(nx)}{n^2}$ converges uniformly on all of $\mathbb{R}$.
</Problem>

<Solution>
*Proof.* For all $x \in \mathbb{R}$ and all $n \geq 1$:

$$\left|\frac{\cos(nx)}{n^2}\right| \leq \frac{|\cos(nx)|}{n^2} \leq \frac{1}{n^2} =: M_n.$$

The series $\sum M_n = \sum 1/n^2 = \pi^2/6 < \infty$ converges (it is a $p$-series with $p = 2 > 1$).

By the Weierstrass $M$-test, $\sum \frac{\cos(nx)}{n^2}$ converges uniformly on $\mathbb{R}$. $\square$
</Solution>

<Problem>
**Exercise 14.3 (Counterexample).** Show that the limit-integral interchange fails without uniform convergence. Define $f_n \colon [0, 1] \to \mathbb{R}$ by

$$f_n(x) = n^2 x \cdot e^{-nx}.$$

1. Show that $f_n(x) \to 0$ pointwise for each $x \in [0, 1]$.
2. Compute $\int_0^1 f_n(x)\,dx$ and show $\lim_{n \to \infty} \int_0^1 f_n(x)\,dx = 1 \neq 0 = \int_0^1 \lim_{n \to \infty} f_n(x)\,dx$.
3. Verify the convergence is not uniform by finding the maximum of $f_n$ on $[0, 1]$.

*Hint:* The maximum of $f_n$ occurs at $x = 1/n$.
</Problem>

<Solution>
*Proof.*

**(1) Pointwise convergence to $0$.** Fix $x \in [0, 1]$. If $x = 0$: $f_n(0) = 0$ for all $n$. If $x > 0$:

$$f_n(x) = n^2 x \cdot e^{-nx} = \frac{n^2 x}{e^{nx}}.$$

Since $e^{nx} \to \infty$ exponentially while the numerator grows polynomially, $f_n(x) \to 0$ for each fixed $x > 0$.

**(2) The integral does not converge to $0$.** Integrate by parts with $u = n^2 x$, $dv = e^{-nx}\,dx$:

$$\int_0^1 n^2 x \, e^{-nx}\,dx = \left[-nx \, e^{-nx}\right]_0^1 + \int_0^1 n \, e^{-nx}\,dx = -ne^{-n} + \left[-e^{-nx}\right]_0^1 = -ne^{-n} + 1 - e^{-n}.$$

As $n \to \infty$: $ne^{-n} \to 0$ and $e^{-n} \to 0$, so

$$\lim_{n \to \infty} \int_0^1 f_n(x)\,dx = 0 + 1 - 0 = 1.$$

But $\int_0^1 \lim f_n(x)\,dx = \int_0^1 0\,dx = 0$. So $\lim \int f_n \neq \int \lim f_n$.

**(3) The convergence is not uniform.** The maximum of $f_n(x) = n^2 x \, e^{-nx}$ occurs where $f_n'(x) = n^2 e^{-nx}(1 - nx) = 0$, i.e., at $x = 1/n$. The maximum value is:

$$f_n(1/n) = n^2 \cdot \frac{1}{n} \cdot e^{-1} = \frac{n}{e}.$$

Since $\sup_{x \in [0,1]} f_n(x) = n/e \to \infty$, the functions $f_n$ are not even uniformly bounded, let alone uniformly convergent. Theorem 14.2 does not apply, and the interchange fails. $\square$
</Solution>

## When Interchange Works and Fails

# === CASE 1: Interchange WORKS (uniform convergence) ===
# f_n(x) = x^n on [0, 0.9] (uniformly convergent since 0.9 < 1)
# Limit function: f(x) = 0 on [0, 0.9]

print("CASE 1: f_n(x) = x^n on [0, 0.9]")
print("Uniform convergence on [0, r] for r < 1")
print("-" * 50)

r = 0.9
for n in [1, 5, 10, 50, 100]:
    # integral_0^r x^n dx = r^(n+1) / (n+1)
    integral_fn = r**(n+1) / (n+1)
    # integral of limit = integral of 0 = 0
    print(f"n = {n:>4}: integral(f_n) = {integral_fn:.8f}")

print(f"Limit of integrals -> 0  (matches integral of limit = 0)")
print()

# === CASE 2: Interchange FAILS (only pointwise, spike example) ===
# f_n is a triangle spike: height n, base 2/n, area = 1
# f_n -> 0 pointwise, but integral(f_n) = 1 for all n

print("CASE 2: Spike functions (pointwise but NOT uniform)")
print("-" * 50)

def spike_integral(n):
    """Integral of spike function = area of triangle = (base * height)/2"""
    base = 2.0 / n
    height = n
    return base * height / 2  # Always equals 1

def spike_at_point(n, x):
    """Evaluate spike function at x"""
    if 0 <= x <= 1.0/n:
        return n * n * x      # rising slope: n^2 * x
    elif x <= 2.0/n:
        return n * n * (2.0/n - x)  # falling slope
    else:
        return 0.0

print(f"{'n':>6} {'integral(f_n)':>15} {'f_n(0.1)':>12} {'f_n(0.01)':>12}")
print("-" * 50)
for n in [1, 5, 10, 50, 100, 500, 1000]:
    integral = spike_integral(n)
    val1 = spike_at_point(n, 0.1)
    val2 = spike_at_point(n, 0.01)
    print(f"{n:>6} {integral:>15.1f} {val1:>12.4f} {val2:>12.4f}")

print()
print("f_n -> 0 pointwise (each fixed x is eventually past the spike)")
print("But integral(f_n) = 1 for ALL n, not 0 = integral(0)")
print("The interchange FAILS because convergence is not uniform.")
print()

# === CASE 3: Term-by-term integration via M-test ===
# sum(cos(nx)/n^2) converges uniformly by M-test
# integral_0^pi sum(cos(nx)/n^2) = sum integral_0^pi cos(nx)/n^2 dx

print("CASE 3: Term-by-term integration of sum(cos(nx)/n^2)")
print("-" * 50)

# Numerical integration by Riemann sum
def riemann_sum(func, a, b, N_pts=10000):
    dx = (b - a) / N_pts
    return sum(func(a + (i+0.5)*dx) * dx for i in range(N_pts))

def partial_series(x, N):
    return sum(math.cos(n*x) / n**2 for n in range(1, N+1))

# Integral of partial sum (direct)
for N in [5, 10, 50, 100]:
    val = riemann_sum(lambda x: partial_series(x, N), 0, math.pi)
    # Exact: sum_{n=1}^N integral_0^pi cos(nx)/n^2 dx
    #      = sum_{n=1}^N sin(n*pi)/(n^3) = 0 for all N
    # (since sin(n*pi) = 0)
    print(f"N = {N:>4}: integral_0^pi S_N(x) dx = {val:.8f}")

print()
print("The integrals are approximately 0, which matches the")
print("term-by-term result: each integral(cos(nx)/n^2, 0, pi)")
print("= sin(n*pi)/n^3 = 0. The M-test justifies the interchange.")
