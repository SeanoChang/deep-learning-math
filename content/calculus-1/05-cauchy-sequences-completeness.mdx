## Motivation

So far, every time we proved a sequence converges, we needed to know (or guess) the limit $L$ in advance. The Cauchy criterion offers a fundamentally different approach: it detects convergence purely from the *internal behavior* of the sequence, without reference to any candidate limit. This is one of the deepest consequences of completeness.

## Cauchy Sequences

<Callout type="definition" title="Cauchy Sequence">
A sequence $(a_n)$ is a **Cauchy sequence** if: for every $\epsilon > 0$, there exists $N \in \mathbb{N}$ such that for all $m, n \geq N$,

$$|a_m - a_n| < \epsilon.$$

In symbols: $\forall \epsilon > 0,\; \exists N \in \mathbb{N},\; \forall m, n \geq N,\; |a_m - a_n| < \epsilon.$
</Callout>

<Callout type="tip" title="Intuition">
A Cauchy sequence is one whose terms become arbitrarily close *to each other* for large indices. The definition says nothing about what value they approach -- only that the mutual distances shrink. The remarkable fact is that in $\mathbb{R}$, this is enough to guarantee convergence.
</Callout>

## Convergent Implies Cauchy

<Callout type="theorem" title="Convergent Implies Cauchy">
Every convergent sequence is a Cauchy sequence.
</Callout>

**Proof.** Suppose $a_n \to L$. Let $\epsilon > 0$. There exists $N$ with $|a_n - L| < \epsilon/2$ for all $n \geq N$. For $m, n \geq N$:

$$|a_m - a_n| = |a_m - L + L - a_n| \leq |a_m - L| + |a_n - L| < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon.$$

Hence $(a_n)$ is Cauchy. $\blacksquare$

<Callout type="info" title="The Hard Direction">
The converse -- every Cauchy sequence converges -- is much deeper and relies on completeness. It fails in $\mathbb{Q}$, as we will see.
</Callout>

## Cauchy Implies Bounded

<Callout type="theorem" title="Cauchy Implies Bounded">
Every Cauchy sequence is bounded.
</Callout>

**Proof.** Let $(a_n)$ be Cauchy. Taking $\epsilon = 1$, there exists $N$ with $|a_m - a_n| < 1$ for all $m, n \geq N$. In particular, for all $n \geq N$:

$$|a_n| = |a_n - a_N + a_N| \leq |a_n - a_N| + |a_N| < 1 + |a_N|.$$

Set $M = \max(|a_1|, |a_2|, \ldots, |a_{N-1}|, 1 + |a_N|)$. Then $|a_n| \leq M$ for all $n$. $\blacksquare$

<Callout type="tip" title="Parallel Structure">
Notice the proof is almost identical to "convergent implies bounded." In both cases, we use the definition to control the tail, then handle finitely many initial terms by brute force. This is a common pattern in sequence arguments.
</Callout>

## The Cauchy Criterion: Every Cauchy Sequence in $\mathbb{R}$ Converges

<Callout type="theorem" title="Cauchy Criterion (Completeness of R)">
A sequence in $\mathbb{R}$ converges if and only if it is a Cauchy sequence.
</Callout>

We already proved ($\Rightarrow$). The key is ($\Leftarrow$): every Cauchy sequence in $\mathbb{R}$ converges.

**Proof ($\Leftarrow$).** Let $(a_n)$ be a Cauchy sequence in $\mathbb{R}$.

**Step 1: $(a_n)$ is bounded** (proved above).

**Step 2: Extract a convergent subsequence.** By the Bolzano-Weierstrass theorem, the bounded sequence $(a_n)$ has a convergent subsequence $a_{n_k} \to L$.

**Step 3: The full sequence converges to $L$.** Let $\epsilon > 0$. Since $(a_n)$ is Cauchy, there exists $N_1$ with $|a_m - a_n| < \epsilon/2$ for all $m, n \geq N_1$. Since $a_{n_k} \to L$, there exists $K$ with $|a_{n_k} - L| < \epsilon/2$ for all $k \geq K$.

Choose $k_0 \geq K$ large enough that $n_{k_0} \geq N_1$ (possible since $n_k \to \infty$). Then for all $n \geq N_1$:

$$|a_n - L| \leq |a_n - a_{n_{k_0}}| + |a_{n_{k_0}} - L| < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon.$$

Therefore $a_n \to L$. $\blacksquare$

<Callout type="theorem" title="Summary: Three Equivalent Statements">
The following are equivalent for a sequence $(a_n)$ in $\mathbb{R}$:

1. $(a_n)$ converges.
2. $(a_n)$ is a Cauchy sequence.
3. $(a_n)$ is bounded and has exactly one subsequential limit.
</Callout>

## Nested Interval Theorem Revisited

The nested interval property provides an alternative route to proving the Cauchy criterion, and highlights the deep connections between the various forms of completeness.

<Callout type="theorem" title="Nested Interval Theorem (Restated)">
If $[a_n, b_n]$ is a nested sequence of closed intervals with $b_n - a_n \to 0$, then $\bigcap_{n=1}^{\infty} [a_n, b_n] = \{c\}$ for a unique $c \in \mathbb{R}$.
</Callout>

The equivalences among completeness, MCT, Bolzano-Weierstrass, the Cauchy criterion, and the nested interval property form the conceptual spine of the real number system:

$$\text{Completeness} \iff \text{MCT} \iff \text{BW} \iff \text{Cauchy criterion} \iff \text{Nested intervals}$$

Each can be derived from any other, using only the ordered field axioms.

## Contrast with $\mathbb{Q}$: Incompleteness

<Callout type="warning" title="Q is Not Complete">
The Cauchy criterion fails in $\mathbb{Q}$. There exist Cauchy sequences of rationals that do not converge in $\mathbb{Q}$. The "missing" limits are precisely the irrational numbers.
</Callout>

**Example.** Define $a_1 = 1$ and

$$a_{n+1} = \frac{a_n}{2} + \frac{1}{a_n}.$$

Each $a_n$ is rational (by induction). The sequence is Cauchy and converges to $\sqrt{2}$ in $\mathbb{R}$. But since $\sqrt{2} \notin \mathbb{Q}$, this Cauchy sequence has no limit in $\mathbb{Q}$.

**Proof that $a_n \to \sqrt{2}$:** Note $a_{n+1} = \frac{1}{2}(a_n + 2/a_n)$. By AM-GM, $a_{n+1} \geq \sqrt{a_n \cdot 2/a_n} = \sqrt{2}$ for all $n$. For $n \geq 2$, the sequence is decreasing (since $a_n \geq \sqrt{2}$ gives $a_{n+1} - a_n = (2 - a_n^2)/(2a_n) \leq 0$). By MCT, $L = \lim a_n$ exists. The recurrence gives $L = L/2 + 1/L$, so $L/2 = 1/L$, hence $L^2 = 2$ and $L = \sqrt{2}$.

<Problem title="Verify: A Sequence is Cauchy">
Let $a_n = \sum_{k=1}^{n} 1/k!$. Show that $(a_n)$ is a Cauchy sequence. (You may use the fact that $k! \geq 2^{k-1}$ for $k \geq 1$.)

<Solution>
**Proof.** For $m > n$:

$$|a_m - a_n| = \sum_{k=n+1}^{m} \frac{1}{k!} \leq \sum_{k=n+1}^{m} \frac{1}{2^{k-1}} = \frac{1}{2^n} \cdot \frac{1 - (1/2)^{m-n}}{1 - 1/2} < \frac{1}{2^n} \cdot 2 = \frac{1}{2^{n-1}}.$$

Given $\epsilon > 0$, choose $N$ with $1/2^{N-1} < \epsilon$ (i.e., $N > 1 + \log_2(1/\epsilon)$). Then for all $m > n \geq N$:

$$|a_m - a_n| < \frac{1}{2^{n-1}} \leq \frac{1}{2^{N-1}} < \epsilon.$$

Hence $(a_n)$ is Cauchy. (By the Cauchy criterion, it converges. Its limit is $e - 1$.) $\blacksquare$
</Solution>
</Problem>

<Problem>
**Exercise 5.3 (Counterexample).** Exhibit a Cauchy sequence in $\mathbb{Q}$ that does not converge in $\mathbb{Q}$. Prove that your sequence is Cauchy and that its limit is not rational.

*Hint:* Use the sequence $a_1 = 1$, $a_{n+1} = a_n/2 + 1/a_n$, which converges to $\sqrt{2}$.
</Problem>

<Solution>
*Answer.* Define $a_1 = 1$ and $a_{n+1} = \frac{a_n}{2} + \frac{1}{a_n}$ for $n \geq 1$.

**Each $a_n$ is rational:** By induction. $a_1 = 1 \in \mathbb{Q}$. If $a_n \in \mathbb{Q}$, then $a_{n+1} = a_n/2 + 1/a_n$ is a sum and quotient of rationals (with $a_n \neq 0$), so $a_{n+1} \in \mathbb{Q}$.

**The sequence converges to $\sqrt{2}$ in $\mathbb{R}$:** Note $a_{n+1} = \frac{1}{2}(a_n + 2/a_n)$. By AM-GM, $a_{n+1} \geq \sqrt{a_n \cdot 2/a_n} = \sqrt{2}$ for all $n \geq 1$, so $a_n \geq \sqrt{2}$ for $n \geq 2$. For $n \geq 2$, $a_{n+1} - a_n = (2 - a_n^2)/(2a_n) \leq 0$, so the sequence is decreasing and bounded below by $\sqrt{2}$. By MCT, $L = \lim a_n$ exists. The recurrence gives $L = L/2 + 1/L$, hence $L^2 = 2$, so $L = \sqrt{2}$.

**Cauchy but no limit in $\mathbb{Q}$:** Since $(a_n)$ converges in $\mathbb{R}$, it is Cauchy. But $\sqrt{2} \notin \mathbb{Q}$, so the sequence has no limit in $\mathbb{Q}$. This demonstrates that $\mathbb{Q}$ is not complete: the Cauchy criterion fails in $\mathbb{Q}$. $\square$
</Solution>

<Problem title="Prove: Cauchy Implies Bounded (Your Turn)">
Reproduce the proof that every Cauchy sequence is bounded. Be explicit about how you handle the finitely many initial terms.

<Solution>
**Proof.** Let $(a_n)$ be Cauchy. Applying the Cauchy definition with $\epsilon = 1$: there exists $N \in \mathbb{N}$ such that $|a_m - a_n| < 1$ for all $m, n \geq N$.

In particular, taking $m = N$: for all $n \geq N$, $|a_n - a_N| < 1$, so by the triangle inequality:

$$|a_n| \leq |a_n - a_N| + |a_N| < 1 + |a_N|.$$

Now set

$$M = \max(|a_1|, |a_2|, \ldots, |a_{N-1}|, 1 + |a_N|).$$

For $n < N$: $|a_n| \leq M$ by definition of the max. For $n \geq N$: $|a_n| < 1 + |a_N| \leq M$.

Therefore $|a_n| \leq M$ for all $n \in \mathbb{N}$, and $(a_n)$ is bounded. $\blacksquare$
</Solution>
</Problem>
