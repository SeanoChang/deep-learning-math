# The Derivative: Definition and First Properties

The derivative is the central object of differential calculus. Where limits gave us a language for describing
approach and approximation, the derivative uses that language to capture the *instantaneous rate of change*
of a function. In this lesson we develop the definition rigorously from the difference quotient, prove its
first structural property (differentiability implies continuity), and examine the geometry of local linearization.

## The Difference Quotient

Given a function $f$ defined on an open interval containing $a$, the **difference quotient** at $a$ with
increment $h \neq 0$ is:

$$\frac{f(a+h) - f(a)}{h}$$

Geometrically this is the slope of the secant line through $(a, f(a))$ and $(a+h, f(a+h))$.

As $h \to 0$, the secant line pivots toward a limiting position -- the **tangent line** -- provided the
limit exists.

## Definition of the Derivative

<Callout type="definition">
**Definition (Derivative).** Let $f$ be defined on an open interval containing $a$. The **derivative of $f$ at $a$** is

$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

provided this limit exists (as a finite real number). If $f'(a)$ exists we say $f$ is **differentiable at $a$**.
Equivalently, using the substitution $x = a + h$:

$$f'(a) = \lim_{x \to a} \frac{f(x) - f(a)}{x - a}.$$

If $f$ is differentiable at every point of an open interval $I$, we say $f$ is **differentiable on $I$**, and
$f' \colon I \to \mathbb{R}$ is the **derivative function**.
</Callout>

## First Examples from the Definition

<Callout type="theorem">
**Proposition (Derivative of a constant).** If $f(x) = c$ for all $x$, then $f'(a) = 0$ for every $a$.

*Proof.* For any $h \neq 0$,

$$\frac{f(a+h) - f(a)}{h} = \frac{c - c}{h} = 0.$$

Hence $\lim_{h \to 0} 0 = 0$. $\blacksquare$
</Callout>

<Callout type="theorem">
**Proposition (Derivative of the identity).** If $f(x) = x$, then $f'(a) = 1$ for every $a$.

*Proof.* $\frac{f(a+h) - f(a)}{h} = \frac{(a+h) - a}{h} = 1$ for all $h \neq 0$. $\blacksquare$
</Callout>

<Callout type="theorem">
**Proposition (Power rule from first principles).** If $f(x) = x^n$ for $n \in \mathbb{N}$, then $f'(a) = na^{n-1}$.

*Proof.* Using the factorization $x^n - a^n = (x - a)(x^{n-1} + x^{n-2}a + \cdots + a^{n-1})$, we have

$$\frac{f(x) - f(a)}{x - a} = x^{n-1} + x^{n-2}a + \cdots + a^{n-1}.$$

This is a sum of $n$ terms. As $x \to a$, each term $\to a^{n-1}$, so the limit is $na^{n-1}$. $\blacksquare$
</Callout>

## Differentiability Implies Continuity

This is one of the most important structural results connecting the two core limit-based concepts.

<Callout type="theorem">
**Theorem (Differentiability implies continuity).** If $f$ is differentiable at $a$, then $f$ is continuous at $a$.

*Proof.* We need to show $\lim_{x \to a} f(x) = f(a)$, i.e., $\lim_{x \to a} [f(x) - f(a)] = 0$.

Write

$$f(x) - f(a) = \frac{f(x) - f(a)}{x - a} \cdot (x - a)$$

for $x \neq a$. Taking the limit as $x \to a$:

$$\lim_{x \to a} [f(x) - f(a)] = \lim_{x \to a} \frac{f(x) - f(a)}{x - a} \cdot \lim_{x \to a}(x - a) = f'(a) \cdot 0 = 0.$$

The first limit exists and equals $f'(a)$ by hypothesis (differentiability), and the second limit is $0$.
By the product rule for limits, the overall limit is $0$. Hence $f$ is continuous at $a$. $\blacksquare$
</Callout>

<Callout type="warning">
**The converse is false.** Continuity does *not* imply differentiability. The function $f(x) = |x|$ is
continuous everywhere but not differentiable at $x = 0$. Even more dramatically, Weierstrass constructed
a function that is continuous everywhere but differentiable *nowhere*.
</Callout>

## Examples of Nondifferentiability

### Corner: $f(x) = |x|$ at $x = 0$

The left-hand and right-hand limits of the difference quotient disagree:

$$\lim_{h \to 0^+} \frac{|h| - 0}{h} = 1, \qquad \lim_{h \to 0^-} \frac{|h| - 0}{h} = -1.$$

Since these one-sided limits differ, $f'(0)$ does not exist.

### Vertical tangent: $f(x) = x^{1/3}$ at $x = 0$

$$\frac{f(h) - f(0)}{h} = \frac{h^{1/3}}{h} = h^{-2/3} \to +\infty.$$

The limit is infinite, so $f$ is not differentiable at $0$ (the tangent line is vertical).

### Cusp: $f(x) = x^{2/3}$ at $x = 0$

The left and right difference quotients both diverge (to $-\infty$ and $+\infty$ respectively).

## Local Linear Approximation

<Callout type="theorem">
**Theorem (Characterization via linear approximation).** $f$ is differentiable at $a$ with derivative $L$
if and only if there exists a function $\epsilon(h)$ with $\lim_{h \to 0} \epsilon(h) = 0$ such that

$$f(a+h) = f(a) + Lh + \epsilon(h) \cdot h.$$

*Proof.* ($\Rightarrow$) Suppose $f'(a) = L$. Define $\epsilon(h) = \frac{f(a+h) - f(a)}{h} - L$ for $h \neq 0$ and $\epsilon(0) = 0$.
Then $\lim_{h \to 0}\epsilon(h) = f'(a) - L = 0$, and rearranging gives $f(a+h) = f(a) + Lh + \epsilon(h)h$.

($\Leftarrow$) Given the representation, divide by $h$:

$$\frac{f(a+h) - f(a)}{h} = L + \epsilon(h) \to L \text{ as } h \to 0. \quad \blacksquare$$
</Callout>

<Callout type="info">
This characterization is the **local linearization** viewpoint: near $a$, the function $f$ is well-approximated
by the affine function $T(x) = f(a) + f'(a)(x - a)$, with error that vanishes faster than $|x - a|$.
This is the beginning of the approximation philosophy that culminates in Taylor's theorem.
</Callout>

## Exercises

<Problem>
**Exercise 1.** Prove from the definition that if $f$ is differentiable at $a$, then $f$ is continuous at $a$.
State clearly where you use the hypothesis of differentiability.
</Problem>

<Solution>
We must show $\lim_{x \to a} f(x) = f(a)$. Write $f(x) - f(a) = \frac{f(x)-f(a)}{x-a} \cdot (x-a)$ for $x \neq a$.

By hypothesis, $f'(a)$ exists, so $\lim_{x \to a}\frac{f(x)-f(a)}{x-a} = f'(a)$ (finite). Also $\lim_{x \to a}(x-a) = 0$.

By the product rule for limits (both limits exist, one is finite):

$$\lim_{x \to a}[f(x) - f(a)] = f'(a) \cdot 0 = 0.$$

Hence $\lim_{x \to a}f(x) = f(a)$, so $f$ is continuous at $a$. The key use of differentiability is
that the limit of the difference quotient exists as a *finite* number, allowing us to apply the product rule for limits.
</Solution>

<Problem>
**Exercise 2.** Let $f(x) = x^{2/3}$. Show that $f$ is not differentiable at $x = 0$ by computing the
difference quotient and showing the limit does not exist (finitely).
</Problem>

<Solution>
The difference quotient at $a = 0$ is:

$$\frac{f(0+h) - f(0)}{h} = \frac{h^{2/3}}{h} = h^{-1/3} = \frac{1}{h^{1/3}}.$$

As $h \to 0^+$, $h^{-1/3} \to +\infty$. As $h \to 0^-$, $h^{-1/3} \to -\infty$.

The one-sided limits are both infinite (and of opposite sign), so the two-sided limit does not exist
as a finite number. Therefore $f$ is not differentiable at $0$.

Geometrically, $f(x) = x^{2/3}$ has a cusp at the origin.
</Solution>

<Problem>
**Exercise 3 (Theorem Application).** Let $f$ and $g$ be differentiable at $a$ with $f(a) = g(a)$ and $f'(a) = g'(a)$. Using the characterization via linear approximation (i.e., $f(a+h) = f(a) + f'(a)h + \epsilon(h)h$), prove that

$$\lim_{h \to 0} \frac{f(a+h) - g(a+h)}{h} = 0.$$

*Hint:* Write each function in its local linear approximation form and subtract.
</Problem>

<Solution>
*Proof.* Since $f$ is differentiable at $a$, there exists $\epsilon_1(h)$ with $\lim_{h \to 0}\epsilon_1(h) = 0$ and

$$f(a+h) = f(a) + f'(a)h + \epsilon_1(h)h.$$

Similarly, since $g$ is differentiable at $a$, there exists $\epsilon_2(h)$ with $\lim_{h \to 0}\epsilon_2(h) = 0$ and

$$g(a+h) = g(a) + g'(a)h + \epsilon_2(h)h.$$

Subtracting:

$$f(a+h) - g(a+h) = [f(a) - g(a)] + [f'(a) - g'(a)]h + [\epsilon_1(h) - \epsilon_2(h)]h.$$

Since $f(a) = g(a)$ and $f'(a) = g'(a)$, this simplifies to $f(a+h) - g(a+h) = [\epsilon_1(h) - \epsilon_2(h)]h$. Dividing by $h$:

$$\frac{f(a+h) - g(a+h)}{h} = \epsilon_1(h) - \epsilon_2(h) \to 0 - 0 = 0. \quad \square$$
</Solution>

<Problem>
**Exercise 4 (Counterexample).** Give an example of a function that is continuous at $x = 0$ but not differentiable at $x = 0$. Prove both claims rigorously from the definitions.

*Hint:* The standard example is $f(x) = |x|$.
</Problem>

<Solution>
*Answer.* Let $f(x) = |x|$.

**Continuous at $0$:** Let $\varepsilon > 0$. Choose $\delta = \varepsilon$. If $|x - 0| = |x| < \delta$, then $|f(x) - f(0)| = ||x| - 0| = |x| < \delta = \varepsilon$. So $f$ is continuous at $0$.

**Not differentiable at $0$:** Consider the difference quotient:

$$\frac{f(0 + h) - f(0)}{h} = \frac{|h|}{h} = \begin{cases} 1 & \text{if } h > 0 \\ -1 & \text{if } h < 0. \end{cases}$$

The right-hand limit is $\lim_{h \to 0^+} |h|/h = 1$ and the left-hand limit is $\lim_{h \to 0^-} |h|/h = -1$. Since $1 \neq -1$, the two-sided limit $\lim_{h \to 0} \frac{|h|}{h}$ does not exist, so $f'(0)$ does not exist.

This shows the converse of "differentiable implies continuous" is false. $\square$
</Solution>

## Numerical vs. Analytical Differentiation

<PythonPlayground
  title="Numerical differentiation vs the analytical derivative"
  defaultCode={`import numpy as np

def f(x):
    return x**3

def f_prime_exact(x):
    return 3*x**2

a = 2.0
print(f"Analytical derivative at x={a}: {f_prime_exact(a)}")
print()

# Forward difference quotient for decreasing h
print("h             | Forward diff  | Error")
print("-" * 50)
for k in range(1, 16):
    h = 10**(-k)
    approx = (f(a + h) - f(a)) / h
    error = abs(approx - f_prime_exact(a))
    print(f"1e-{k:<10} | {approx:<13.8f} | {error:.2e}")

print()
print("Notice: error decreases until ~h=1e-5,")
print("then floating-point cancellation causes it to grow!")
`}
/>

<Callout type="tip">
The playground above demonstrates an important computational phenomenon: the **forward difference quotient**
converges as $h \to 0$ in exact arithmetic, but on a computer, taking $h$ too small causes catastrophic
cancellation in $f(a+h) - f(a)$. The **central difference** $\frac{f(a+h) - f(a-h)}{2h}$ has better
numerical properties. Try modifying the code to compare!
</Callout>
