# The Rigorous AI Learning Path
## Twelve 15-Week Course Plans — From Optimization to Alignment

> **Path philosophy:** The correct way to organize AI is not by hype labels or course titles, but by dependency of concepts. Each course in this sequence assumes mastery of all prior courses. The conceptual spine is:
>
> optimization → statistical learning → probabilistic modeling → representation learning → architectures → domains → decision-making / generation → systems / scale / robustness

**Prerequisites assumed:** Strong calculus (multivariable + real analysis intuition), linear algebra (through spectral theory), and probability (measure-theoretic intuition helpful but not required).
